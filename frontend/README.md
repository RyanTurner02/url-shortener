# Frontend

This is the frontend for the URL Shortener web app, built with **React** and **Next.js**.

## Prerequisites

* Node.js 24+
* VS Code

## Debugging

1. Open **VS Code**
2. Select **Open Folder...**
3. Open `url-shortener/frontend`
4. Open the terminal (**Terminal** > **New Terminal**)
5. Install dependencies:

```bash
npm install
```

6. Go to the **Run and Debug** tab
7. Start the **Next.js: debug full stack** configuration

## Testing

1. Install the [**Vitest**](https://marketplace.visualstudio.com/items?itemName=vitest.explorer) VS Code extension
2. Go to the **Testing** tab
3. Select tests
4. Click **Run Test** or **Debug Test**
